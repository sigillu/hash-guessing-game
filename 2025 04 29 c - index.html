üîê What is a hash?

A hash is a special kind of digital fingerprint. It takes any input ‚Äî like a word or a file ‚Äî and transforms it into a unique string of characters. If you change even a single letter, the resulting hash changes completely. This process is one-way, meaning there's no way to go backward and figure out what the original input was just by looking at the hash.

üéØ What did this game teach you?

By giving you the hash of a secret word, the game challenged you to find the original word. But since hashes can't be reversed, your only strategy was to guess possible words, hash them, and compare the results. When a guess matched the target hash, you knew you'd found the original word.

This hands-on experience illustrates a core principle of hashing: you can verify a match, but you can't work backward to reveal the secret.

üîç Where are hashes used in real life?

  • Crypto Mining (Proof of Work): In blockchains like Bitcoin, miners guess different values (nonces) to find a hash that meets specific rules. It‚Äôs all about trial and error ‚Äî just like this game.

  • Password Storage: Websites don‚Äôt store your actual password. They store its hash. When you log in, the system hashes your input and checks it against the stored hash. That way, even if the database is compromised, your real password stays safe.
